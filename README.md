# Volta Privacy Policy

## Overview

Volta does not collect, store, transmit, or share any personal or non-personal data from the user’s device.

## Data Collection and Use

All data created or used by Volta remains entirely on the user’s device.
The app does not:
	•	Send data to the developer
	•	Send data to external servers
	•	Use analytics, tracking, or advertising services
	•	Collect crash reports or diagnostic data

The developer has no access to any data generated or stored by the app.

## Third-Party Services

Volta does not use any third-party services that collect user data.

## Children’s Privacy

Volta does not knowingly collect any personal information from children or any users.

## Changes to This Privacy Policy

If this Privacy Policy is updated, changes will be reflected on this page.
